# Analisis-de-negocio
## Showz-Marketing-Analytics-Business-Optimization
## 📝 Project Description

This project analyzes customer behavior, sales performance, and marketing efficiency for Showz, an online event ticketing company.

The analysis is based on datasets containing website visits, customer orders, and marketing expenses from January 2017 to December 2018.

The objective is to understand how users interact with the platform, how they convert into paying customers, and whether the company's marketing investments generate profitable returns. The insights help optimize the marketing budget by identifying the most effective acquisition channels.

## 🎯 Project Objective

The main objective of this project is to evaluate customer acquisition, purchasing behavior, and marketing performance to determine where marketing resources should be invested.

To achieve this, the analysis includes:

Analyzing customer website usage and engagement
Measuring customer conversion from first visit to first purchase
Calculating Customer Lifetime Value (LTV)
Estimating Customer Acquisition Cost (CAC)
Evaluating Return on Marketing Investment (ROMI)
Comparing marketing channels to identify the most profitable acquisition sources
Providing data-driven recommendations for marketing budget allocation

This type of analysis enables companies to optimize customer acquisition strategies, improve marketing efficiency, and maximize business profitability.
